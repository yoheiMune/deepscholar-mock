# deepscholar-mock
Mockup for DeepScholar.

## Requirements.
### Versions.
```
$ node --version
v8.9.2 LTS

$ npm --version
5.5.1
```
### Libraries.
```
$ npm install
```

## How to start.
```
$ node app.js

# Or
$ node-dev app.js
```
